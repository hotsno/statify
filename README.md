# Statify 
Game stat Discord bot with support for multiple games.

# Commands
Use the prefix `.s` for all commands.

`.s fortnite` takes 3 parameters seperated by commas. 1) Epic username. 2) Platform (pc, psn, xbl). 3) Gamemode (solo, duo, squad, lifetime).
